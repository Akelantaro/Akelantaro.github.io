Данный проект представляет собой блок протитипа Progressbar который выполняет 3 функции:
  1 Поле ввода для указания размера дуги кольца в процентах от 0 до 100
  2 Переключатель анимации при активации которого кольцо начинает вращаться вокруг своего центра по часовой стрелке
  3 Переключатель скрытия который скрывает весь блок со страницы оставляя маленький переключатель в верхнем левом углу Прогресс бара для обратного включения
Также он адаптирован под размер устройства и ориентацию экрана и динамически меняет ее при вращении мобильного телефона

в index.html расположена блочная структура с использованием div, а также p input svg и Span
именются указатели на таблицу стилей и файл скриптов

в index.css описана структурная схема вложенных дивов, изменение вида переключателей а также анимации и позиционирование элементов

в index.js описаны скрипты исполнения трех вышеописанных функций, а также адаптация progressbar под размер и ориентацию экрана
